# The BRICK GAME SIMULATOR 9999 in 1
This project - an attempt to recreate the "Brick Game" console on a modern PC as accurately as possible.

## Available games
* Dancer
* Tanks
* Arcanoid
* Racing
* Gun game
* Snakes
* Frogger
* Invaders
* Tetris
* Pentix
* more coming soon...
